# Next.js Admin UI (defuss clone)

A minimal Next.js + shadcn/ui clone of the defuss `with-admin-ui` example.

## Routes
- `/` Login screen
- `/dashboard`
- `/users`
- `/tenants`
- `/api-keys`

## Development
```bash
pnpm dev
```

## Notes
- Uses mock data in `lib/mock-data.ts`.
- Styling is Tailwind v4 + shadcn/ui primitives.
