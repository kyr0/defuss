What does this reset do?
  1. Universal Box Sizing and Margin Reset: Ensures consistent sizing by applying box-sizing: border-box to all elements and removes default margins for cleaner layouts.
  2. Full-Height Layout: Applies height: 100% to html and body elements so they span the full viewport height.
  3. HTML Adjustments: Prevents automatic text resizing in browsers using text-size-adjust, smooths scrolling with scroll-behavior: smooth, and supports light/dark themes with color-scheme.
  4. Typography Enhancements: Improves readability with balanced text wrapping for headings (text-wrap: balance) and ensures friendly line breaks for paragraphs (text-wrap: pretty).
  5. Form Element Inheritance: Ensures form elements (inputs, buttons, etc.) inherit font styles using font: inherit.
  6. Media Element Layout: Media elements like images, videos, and SVGs are displayed as block elements and constrained to their containers with max-width: 100%.
  7. Consistent Decoration: Removes default borders around linked images with a img { border: none }.
  8. Accessibility Focus: Improves focus visibility with :focus styles and honors user preferences for reduced motion using @media (prefers-reduced-motion).
  9. Responsive Typography: Uses clamp() to ensure font sizes scale with the viewport while respecting minimum and maximum sizes, e.g., html { font-size: clamp(12px, 1vw + 1rem, 18px); }.
