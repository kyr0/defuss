## 🛠️ Setup

### 1. Get a decent package manager

We recommend using `bun` as a package manager. It's fast, mature and handles monorepos well. If you haven't installed `bun` yet:

```bash
npm i -g bun@^1.3.9
```

### 2. Install the projects dependencies

```bash
bun i --frozen
```


### 3. Run the development server

```bash
bun run dev
```

### 4. Serve for production

First [preview](https://docs.astro.build/en/develop-and-build/#build-and-preview-your-site) then [deploy](https://docs.astro.build/en/guides/deploy/) your site.
