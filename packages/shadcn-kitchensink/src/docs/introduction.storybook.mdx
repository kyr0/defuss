# defuss-shadcn Component Library

Welcome to the **defuss-shadcn** component library documentation.

## Overview

`defuss-shadcn` provides composable UI primitives built on **Tailwind 4** and **Basecoat UI** conventions. All components work with defuss' real DOM morphing engine - no virtual DOM overhead.

## Key Features

- **Real DOM**: Components render directly to the DOM, no reconciliation needed
- **Composable**: Build complex UIs by combining simple primitives
- **Accessible**: ARIA attributes and keyboard navigation built-in
- **Themeable**: Full dark mode support with CSS custom properties
- **Lightweight**: No framework bloat - just the components you use

## Getting Started

Install the library:

```bash
bun add defuss-shadcn
```

Import Tailwind and Basecoat styles in your app stylesheet:

```css
@import "tailwindcss";
@import "defuss-shadcn/css";
```

Then use components in your JSX:

```tsx
import { Button, Card, CardHeader, CardTitle, CardContent } from "defuss-shadcn";

const App = () => (
  <Card className="max-w-md">
    <CardHeader>
      <CardTitle>Hello defuss-shadcn</CardTitle>
    </CardHeader>
    <CardContent>
      <Button>Get Started</Button>
    </CardContent>
  </Card>
);
```
