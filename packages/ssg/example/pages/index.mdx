Welcome to the defuss-ssg example website! We can use dynamic variables! Look: {someData.firstName} {someData.lastName} has been read from <code>../data.json</code> at generation time.

Currently we can render buttons, but the client-side interactivity is not yet implemented:

<button type="button" class="secondary" onClick={() => alert("Button clicked!")}>Button</button>

We also have **infinite** layout options, because you can mix Markdown and HTML:

## Grid layout

<div class="grid">
  <div>1</div>
  <div>2</div>
  <div>3</div>
  <div>4</div>
</div>

## Math support 

We can use KaTeX for math rendering, both inline $( L = \frac{1}{2} \rho v^2 S C_L )$ and block:

$$
  L = \frac{1}{2} \rho v^2 S C_L.
$$
